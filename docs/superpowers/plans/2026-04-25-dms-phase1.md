**Goal (revised):** 연구원이 자기 과제의 모든 연구산출물 문서를 **번호/버전/유효본/이력 기준으로 10초 안에 신뢰성 있게 찾을 수 있는** 단일 진실 시스템을 8주 내에 구축한다. Phase 1 핵심 = (1) 자동 채번 + UNIQUE 보장 (엑셀 사고 0건), (2) 부모-자식 lineage + 유효본 식별 (Effective/Superseded), (3) 전문 검색 (Postgres FTS + pg_trgm), (4) AD 인증 + 단순 e-결재 (작성/검토/승인 라인 + 결재 결과 PDF 스탬프), (5) LibreOffice Word→PDF 변환 + 일반 MinIO 저장.

**Phase 2로 연기 (C2 적용):** SHA-256 해시체인 audit (Tasks 2/3/4 → 일반 audit_logs로 대체), PIN (Task 11), 동적 워터마크 (Task 13), MinIO Object Lock/WORM (Task 14의 lock 부분), 다중 서명블록 다단 임베드 (Task 13의 signer.py), 인쇄 시 워터마크 재적용 (Task 16), 수동 위임 (Task 15), BCP 종이 양식 (Task 19의 BCP 부분), 해시체인 검증 CLI (Task 4). Phase 1에서는 일반 `audit_logs` 테이블 (append-only는 앱 규약 + revoke INSERT-only grant로만; DB 트리거/해시체인 없음). 21 CFR Part 11 표현은 plan 전체에서 제거하고 "사내 e-결재 + audit log"로만 포지셔닝.

**Architecture:** FastAPI 모놀리스 + Next.js SPA + Postgres + MinIO(일반 버킷) + LibreOffice headless를 단일 Docker Compose로 묶는다. 상태 변경 시 `audit_logs` 테이블에 (actor, action, target, payload_json, ts) append. `documents`, `approvals`, `numbering_sequences` 는 정규 트랜잭션 모델. 최종 승인 시 LibreOffice가 PDF로 변환하고 마지막 페이지에 결재자 이름·시각 단순 스탬프를 append하여 MinIO에 저장 (Object Lock 미사용).

**Tech Stack:** Python 3.12 / FastAPI / SQLAlchemy 2.0 / Alembic / Postgres 16 (FTS + pg_trgm) / Next.js 14 (App Router) / TypeScript / TanStack Query / python-ldap3 / python-docx / lxml / LibreOffice 24 (headless) / MinIO (일반) / reportlab + pypdf / pytest / Docker Compose

## Context

현재 사기업 연구소에서 연구산출물 문서를 다음과 같이 운영 중:

- 문서번호를 우리 팀이 수동 채번 → 과제별 엑셀 시트에 목록 관리 (사고/중복 발생)
- 종이 문서 + 수기 서명 결재 → 결재권자 직접 방문 부담
- 승인본은 문서고에 종이 보관, 대출 요청 시 직접 인출
- 별도 스캔본은 OCR 미적용 → 검색 곤란

이를 종이 0%로 완전 전자화하고, GxP/Data Integrity(ALCOA+)를 자체 그레이드로 지향하면서, 워크플로우/비용 사유로 자체 구축한다. 입력이 디지털 Word이므로 OCR/스캔 파이프라인은 불필요. 21 CFR Part 11 §11.10/§11.50/§11.70/§11.200 요건은 PKI 없이 (AD ID/PW + PIN + 서명의미 동의 + 결과 PDF SHA-256 stamp) 충족 가능한 수준으로 설계.

```python
from sqlalchemy import BigInteger, String, JSON, Integer

class Event(Base):

    target_id: Mapped[str] = mapped_column(String(128), index=True)
    payload_json: Mapped[dict] = mapped_column(JSON)

    this_hash: Mapped[str] = mapped_column(String(64), unique=True)
```

```python
# alembic/versions/0001_events.py — upgrade()
op.create_table("events",
    sa.Column("id", sa.BigInteger, primary_key=True, autoincrement=True),
    sa.Column("ts_unix", sa.BigInteger, nullable=False),
    sa.Column("actor_id", sa.String(64), nullable=False),
    sa.Column("action_type", sa.String(64), nullable=False),
    sa.Column("target_id", sa.String(128), nullable=False),
    sa.Column("payload_json", sa.JSON, nullable=False),
    sa.Column("prev_hash", sa.String(64), nullable=False),
    sa.Column("this_hash", sa.String(64), nullable=False, unique=True),
)
op.execute("""
CREATE OR REPLACE FUNCTION events_block_modify() RETURNS trigger AS $$
BEGIN RAISE EXCEPTION 'events is append-only'; END;
$$ LANGUAGE plpgsql;
CREATE TRIGGER trg_events_no_update BEFORE UPDATE ON events
  FOR EACH ROW EXECUTE FUNCTION events_block_modify();
CREATE TRIGGER trg_events_no_delete BEFORE DELETE ON events
  FOR EACH ROW EXECUTE FUNCTION events_block_modify();
""")
```

```python
# tests/test_event_log.py
from app.audit.event_log import append_event
from app.audit.hash_chain import compute_hash, GENESIS

def test_first_event_links_to_genesis(db):
    e = append_event(db, actor="u1", action="DOC_SUBMIT", target="d1", payload={"x":1})
```

```python
import hashlib, json
GENESIS = "0" * 64

def compute_hash(prev_hash: str, actor: str, ts: int, action: str, target: str, payload: dict) -> str:
    canonical = json.dumps(payload, sort_keys=True, separators=(",", ":"), ensure_ascii=False)
    msg = f"{prev_hash}|{actor}|{ts}|{action}|{target}|{canonical}".encode("utf-8")
    return hashlib.sha256(msg).hexdigest()
```

```python
def append_event(db, *, actor: str, action: str, target: str, payload: dict) -> Event:
    db.execute(text("SELECT pg_advisory_xact_lock(:k)"), {"k": LOCK_KEY})
    last = db.query(Event).order_by(Event.id.desc()).first()

              payload_json=payload, prev_hash=prev, this_hash=h)

    return e
```

```python
import zipfile, io
from lxml import etree

CP_NS = "http://schemas.openxmlformats.org/officeDocument/2006/custom-properties"
VT_NS = "http://schemas.openxmlformats.org/officeDocument/2006/docPropsVTypes"

def inject_custom_properties(docx_bytes: bytes, props: dict[str,str]) -> bytes:
    src = io.BytesIO(docx_bytes); dst = io.BytesIO()
    with zipfile.ZipFile(src) as zin, zipfile.ZipFile(dst, "w", zipfile.ZIP_DEFLATED) as zout:
        names = set(zin.namelist())
        for n in names:
            if n == "docProps/custom.xml": continue
            zout.writestr(n, zin.read(n))
        # build custom.xml
        root = etree.Element(f"{{{CP_NS}}}Properties", nsmap={"": CP_NS, "vt": VT_NS})
        for i, (k, v) in enumerate(props.items(), start=2):
            p = etree.SubElement(root, f"{{{CP_NS}}}property",
                fmtid="{D5CDD505-2E9C-101B-9397-08002B2CF9AE}", pid=str(i), name=k)
            etree.SubElement(p, f"{{{VT_NS}}}lpwstr").text = v
        zout.writestr("docProps/custom.xml",
            b'<?xml version="1.0" encoding="UTF-8" standalone="yes"?>\n' + etree.tostring(root))
        # patch [Content_Types].xml + _rels/.rels (override Type + relationship)
        # ... (생략 없이 구현 필요)
```

```python
import re
LOCK_BASE = 0xC0FFEE_00000000

def _period_for(pattern: str, now: datetime) -> str:
    if "{year}" in pattern: return f"{now.year}"
    if "{ym}" in pattern: return f"{now.year}-{now.month:02d}"
    return "all"

def assign_number(db, project_id: int, doc_type_id: int, *, now=None) -> str:
    now = now or datetime.utcnow()
    dt = db.get(DocumentType, doc_type_id); pj = db.get(Project, project_id)
    period = _period_for(dt.numbering_pattern, now)
    key = (project_id * 1_000_003 + doc_type_id) ^ hash(period)
    db.execute(text("SELECT pg_advisory_xact_lock(:k)"), {"k": LOCK_BASE | (key & 0xFFFFFFFF)})
    seq = db.query(NumberingSequence).filter_by(
        project_id=project_id, doc_type_id=doc_type_id, period=period).with_for_update().first()
    if not seq:
        seq = NumberingSequence(project_id=project_id, doc_type_id=doc_type_id, period=period, next_seq=1)
        db.add(seq); db.flush()
    n = seq.next_seq
    seq.next_seq += 1
    return dt.numbering_pattern.format(
        project_code=pj.code, doc_type=dt.code, year=now.year,
        ym=f"{now.year}-{now.month:02d}", seq=n)
```

```python
# app/auth/pin.py
import bcrypt
def set_pin(db, user, pin: str):
    user.pin_hash = bcrypt.hashpw(pin.encode(), bcrypt.gensalt()).decode()
    user.pin_fail_count = 0; user.pin_locked_until = None

def verify_pin(db, username: str, pin: str) -> bool:
    u = db.query(User).filter_by(username=username).one()
    if u.pin_locked_until and u.pin_locked_until > datetime.utcnow(): return False
    ok = bcrypt.checkpw(pin.encode(), u.pin_hash.encode())
    if ok: u.pin_fail_count = 0
    else:
        u.pin_fail_count += 1
        if u.pin_fail_count >= 5:
            u.pin_locked_until = datetime.utcnow() + timedelta(minutes=15)
    db.flush()
    return ok
```

```python
from io import BytesIO
from reportlab.pdfgen import canvas
from reportlab.lib.pagesizes import A4
from pypdf import PdfReader, PdfWriter

def _overlay(text: str, n_pages: int) -> bytes:
    buf = BytesIO(); c = canvas.Canvas(buf, pagesize=A4)
    for _ in range(n_pages):
        c.saveState(); c.setFillGray(0.85); c.setFont("Helvetica", 36)
        c.translate(300, 400); c.rotate(45); c.drawCentredString(0, 0, text)
        c.restoreState(); c.showPage()
    c.save(); return buf.getvalue()

def apply_watermark(pdf_bytes: bytes, *, user: str, ts: str, ip: str) -> bytes:
    reader = PdfReader(BytesIO(pdf_bytes))
    overlay = PdfReader(BytesIO(_overlay(f"{user} {ts} {ip}", len(reader.pages))))
    writer = PdfWriter()
    for i, page in enumerate(reader.pages):
        page.merge_page(overlay.pages[i]); writer.add_page(page)
    out = BytesIO(); writer.write(out); return out.getvalue()
```

```python
    if a.status != "Pending": raise ValueError("이미 결재됨")
```

```python
from fastapi import APIRouter, Depends, Query
from sqlalchemy import text
router = APIRouter()

@router.get("/documents/search")
async def search(q: str = Query(..., min_length=1),
                 effective_only: bool = False,
                 project_code: str | None = None,
                 limit: int = 50,
                 user=Depends(current_user), db=Depends(get_db)):
    sql = """
      SELECT d.id, d.doc_number, d.revision, d.title, d.author_username,
             d.effective_status, d.effective_date,
             ts_rank(d.search_tsv, plainto_tsquery('simple', :q)) AS rank,
             similarity(d.doc_number || ' ' || d.title, :q) AS sim
      FROM documents d
      JOIN projects p ON p.id = d.project_id
      WHERE (d.search_tsv @@ plainto_tsquery('simple', :q)
             OR d.doc_number ILIKE '%' || :q || '%'
             OR d.title ILIKE '%' || :q || '%')
    """
    params = {"q": q, "limit": limit}
    if effective_only:
        sql += " AND d.effective_status = 'Effective'"
    if project_code:
        sql += " AND p.code = :pc"; params["pc"] = project_code
    sql += " ORDER BY rank DESC, sim DESC LIMIT :limit"
    rows = db.execute(text(sql), params).mappings().all()
    return {"results": [dict(r) for r in rows]}
```

```python
@router.get("/documents/{doc_id}/lineage")
async def lineage(doc_id: int, user=Depends(current_user), db=Depends(get_db)):
    # 부모 방향 + 자식 방향 모두 수집
    target = db.get(Document, doc_id)
    if not target: raise HTTPException(404)
    chain = []
    cur = target
    while cur.parent_document_id:
        cur = db.get(Document, cur.parent_document_id); chain.insert(0, cur)
    chain.append(target)
    children = db.query(Document).filter_by(parent_document_id=target.id).order_by(Document.id).all()
    chain.extend(children)
    return {"chain": [{"id":d.id,"doc_number":d.doc_number,"revision":d.revision,
                       "title":d.title,"effective_status":d.effective_status,
                       "effective_date":str(d.effective_date) if d.effective_date else None,
                       "relation_type":d.relation_type} for d in chain]}
```

### Top concerns (양 모델 합의)

**[CRITICAL-1] Sponsor 미확정 상태에서 코딩 착수 — 구조적 결함.** Open Q4 "결재권자 3명 식별"은 Phase 1 진행 중 해소가 아니라 **착수 전 hard gate**. e-결재는 결재권자 행동변화를 강제하는 시스템이므로 sponsor 없이 시작 = 권위 없음.

**[CRITICAL-2] Phase 1이 PoC가 아니라 mini-완제품.** 8주에 100% paperless + Part 11 + WORM + 해시체인 + AD + LibreOffice + 워터마크 + 다중서명 + BCP를 다 넣음. 실패 시 원인 분리 불가능. 1-3인 IT팀 기준 16~20주 분량.

**[CRITICAL-3] 프레이밍 오류 — 진짜 병목은 e-sign이 아닌 '단일 진실'.** Context의 통증 순서: ① 채번 사고/중복(운영중단) ② 결재권자 직접 방문(속도) ③ 종이 보관/검색(편의). GxP/Part 11/해시체인은 IT팀 self-imposed. 사기업 R&D(미규제)에서 6개월 후 "왜 FDA처럼 굴고 있지?" 회의 부름. **10x reframing**: "연구산출물의 번호/버전/승인상태/유효본을 누구나 10초 안에 신뢰" → 핵심 제품은 채번/lineage/검색이지 e-signature가 아님.

**[HIGH-4] "100% paperless day 1"은 acceptance criteria로 위험.** 외부 공동연구/legacy 양식/HWP/wet signature/결재권자 부재 등 예외 필연. 한 번의 예외가 시스템 실패처럼 보임. 더 나은 framing: "지정 doc_type 2-3개 default digital".

**[HIGH-5] Part 11 "충족 가능한 수준" — 위험한 자기 선언.** 사기업 R&D는 본인이 외부 감사 받지 않는 이상 검증 불가. 6개월 후 외부 CRO 협업/해외 출원/M&A 실사 시 답 불가. 기 발급 PDF 소급 재서명 불가. 택1: (A) "Part 11/GxP" 표현 제거하고 "사내 e-결재 + audit log"로만 포지셔닝, (B) Phase 1부터 PyHanko+KISA TSA 도입.

**[HIGH-6] Build-vs-Buy 비교 부재.** SharePoint+Power Automate / M-Files / Veeva / OnlyOffice 3년 TCO + 운영부담 + 컴플라이언스 신뢰성 1-pager 비교 필요. "비용 사유" 단독으로는 6개월 후 임원 한마디에 무너짐.

**[MEDIUM-7] ROI 지표가 임원 설득에 부적합.** 채번 시간/lead time/NPS = IT 효율성. VP 관점 강한 지표:
