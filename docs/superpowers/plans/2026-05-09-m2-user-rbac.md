# DMS Milestone 2 — User & RBAC Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the User & RBAC layer — full Admin CRUD for users/roles/permissions, 3D RBAC permission evaluator, validity-window enforcement, single-session enforcement (verify M1 wiring), Auditor expiry schedulers, and a minimal admin UI — so OQ-USER-001..009 are runnable as integration tests against the deployed API.

**Architecture:** Spring Security session-cookie auth (no JWT) with CSRF cookie + `X-XSRF-TOKEN`. All `@Audited` entity changes flow through Hibernate Envers — every new/modified column on `users`/`roles` requires a matching `*_AUD` migration. Audit writes go exclusively through `AuditService` using the `audit_role` DB principal with SHA-256 hash chain. Frontend uses inline styles only (no UI library) matching the `LoginView.vue` pattern. Email is a stub (interface + log impl) until M9.

**Tech Stack:** Spring Boot 3.3.11 / Java 21 / Spring Security 6 / Spring Data JPA / Hibernate Envers / PostgreSQL 16 / Flyway 10 / Testcontainers / JUnit 5 / Vue 3.5 / Vite / TypeScript strict / Pinia / vue-router.

**M2 가 끝나면 무엇이 되어야 하는가**

- 모든 권한 검사가 `@PreAuthorize("hasRole('ADMIN')")` 또는 `PermissionEvaluator` 를 통해 서버 측에서 강제된다 (BR-ACC-004 즉시 효력 보장 — 캐시 없음).
- 단일 세션 강제 (M1 wiring 검증), AccountUnlock 60초 스케줄러, AuditorExpirySession/Warning 일일 cron 이 운영 중이다.

- 모든 사용자/역할/권한 변경이 `audit_logs` 해시체인에 INSERT-only 로 기록되며, Hibernate Envers `*_AUD` 테이블에도 entity 히스토리가 쌓인다.
- OQ-USER-001~009 가 Testcontainers IT 로 실행 가능하고 모두 green 이다.

**스펙 편차 (Plan 단계에서 확정, 검토 시 반영)**

1. `user_id` 정규식: `^[a-zA-Z0-9._-]{2,50}$` — FS BR-USER-001 의 underscore 허용을 정확히 반영 (openapi.yaml `^[a-zA-Z0-9.-]+$` 와 차이; openapi 도 함께 갱신).
2. `users.last_login_at TIMESTAMPTZ` 신규 컬럼 + Envers `users_aud` 동등 컬럼 (DS Deviation §4.2.1).
3. `PUT /admin/roles/{roleId}` — `description` 과 `role_name` 만 수정 허용. `role_code`, `is_system` 은 시드 무결성을 위해 immutable.
4. `GET /admin/users/export?format=csv` — `v_access_review` 결과를 `text/csv` 로 반환 (분기 접근검토 SOP-USER-001 §6.3 대응).
5. `permissions` 테이블 PG16 `UNIQUE NULLS NOT DISTINCT(role_id, category_id, department)` — 조직 전체 권한 중복 방지 (DS §6.7.1).
6. PermissionEvaluator: M2 범위에서 `hasRole` 만 활성. `hasPermission(target, perm)` / `hasPermission(targetId, targetType, perm)` 은 stub (return false). 문서 단위 평가는 M3 에서 활성.
7. AuditAction enum 확장: `USER_CREATED`, `USER_UPDATED`, `USER_DISABLED`, `USER_PASSWORD_RESET`, `ROLE_ASSIGNED`, `ROLE_REVOKED`, `ROLE_UPDATED`, `PERMISSION_GRANTED`, `PERMISSION_UPDATED`, `PERMISSION_REVOKED`, `ACCOUNT_AUTO_UNLOCKED`, `AUDITOR_EXPIRED`, `AUDITOR_EXPIRY_WARNING`, `USER_LAST_LOGIN_UPDATED`.
8. LocalAuthProvider 확장: `valid_from <= today` 그리고 `today <= valid_until + 1 day` (포함 자정) 검증, 성공 시 `last_login_at = now()` 업데이트.
9. 16자 secure-random 임시 비밀번호 생성기 (대/소/숫자/특수 보장, BR-USER-008).

## Implementation Order Rationale

1. DB migrations first (V6 last_login_at, V7 v_access_review, V8 PG16 NULLS NOT DISTINCT) so JPA `ddl-auto: validate` keeps passing.
2. Matching Envers `_AUD` migrations alongside.
3. Entities → Repositories → Enum extension → Email stub (no business logic dependencies).
4. Services in dependency order: UserAdminService (creates) → RoleAdminService → PermissionAdminService → AccessReviewService.
5. Schedulers and LocalAuthProvider extension before controllers (they use the schedulers' invariants).
6. Controllers next.
7. Security wiring (`@PreAuthorize`, PermissionEvaluator) last so all controller endpoints exist to be guarded.
8. ITs continuously alongside each service/controller (TDD red-green-refactor).
9. Spec doc updates (OQ, openapi, DS) at end so they reflect what shipped.
10. Frontend after backend stabilizes.

```sql
-- backend/src/main/resources/db/migration/V7__access_review_view.sql
-- DS Deviation §6.7.1: v_access_review for SOP-USER-001 quarterly access review.

CREATE OR REPLACE VIEW v_access_review AS
SELECT
    u.id                AS user_pk,
    u.user_id           AS user_id,
    u.full_name         AS full_name,
    u.email             AS email,
    u.department        AS department,
    u.title             AS title,
    u.status            AS status,
    u.valid_from        AS valid_from,
    u.valid_until       AS valid_until,
    u.last_login_at     AS last_login_at,
    u.created_at        AS created_at,
    string_agg(r.role_code, ',' ORDER BY r.role_code) AS role_codes
FROM users u
LEFT JOIN user_roles ur ON ur.user_id = u.id
LEFT JOIN roles r ON r.id = ur.role_id
GROUP BY u.id;

GRANT SELECT ON v_access_review TO app_role;
```

```java
    @Transactional
    public UserDto updateRoles(Long userPk, List<String> roleCodes, String actorUserId, String clientIp) {
        if (roleCodes == null || roleCodes.isEmpty()) {
            throw new UnprocessableEntityException("USER_004", "at least one role required");
        }
        User u = userRepo.findById(userPk).orElseThrow(() -> new NotFoundException("user not found"));
        Set<Role> target = resolveRoles(roleCodes);
        Set<String> targetCodes = new TreeSet<>();
        for (Role r : target) targetCodes.add(r.getRoleCode());

        Set<String> existing = new TreeSet<>();
        for (UserRole ur : u.getRoles()) existing.add(ur.getRole().getRoleCode());

        // revoke removed
        Iterator<UserRole> it = u.getRoles().iterator();
        while (it.hasNext()) {
            UserRole ur = it.next();
            if (!targetCodes.contains(ur.getRole().getRoleCode())) {
                String code = ur.getRole().getRoleCode();
                em.remove(ur);
                it.remove();
                audit.log(new AuditEvent(actorUserId, AuditAction.ROLE_REVOKED, "USER_ROLE",
                        u.getUserId() + ":" + code, null, null,
                        null, clientIp, OffsetDateTime.now(ZoneOffset.UTC)));
            }
        }
        // assign new
        for (Role r : target) {
            if (!existing.contains(r.getRoleCode())) {
                UserRole ur = new UserRole();
                ur.setUser(u);
                ur.setRole(r);
                ur.setAssignedAt(OffsetDateTime.now());
                em.persist(ur);
                audit.log(new AuditEvent(actorUserId, AuditAction.ROLE_ASSIGNED, "USER_ROLE",
                        u.getUserId() + ":" + r.getRoleCode(),
                        null, jsonOf(Map.of("role_code", r.getRoleCode())),
                        null, clientIp, OffsetDateTime.now(ZoneOffset.UTC)));
            }
        }
        em.flush();
        terminateSessions(u.getUserId());
        return UserDto.fromEntity(u);
    }

    @Transactional
    public void disable(String targetUserId, String reason, String actorUserId, String clientIp) {
        if (Objects.equals(targetUserId, actorUserId)) {
            throw new UnprocessableEntityException("USER_005", "you cannot disable your own account (self-disable)");
        }
        User u = userRepo.findByUserId(targetUserId).orElseThrow(() -> new NotFoundException("user not found"));
        u.setStatus(UserStatus.DISABLED);
        userRepo.save(u);

        audit.log(new AuditEvent(actorUserId, AuditAction.USER_DISABLED, "USER",
                String.valueOf(u.getId()),
                jsonOf(Map.of("status", "ACTIVE")),
                jsonOf(Map.of("status", "DISABLED")),
                reason, clientIp, OffsetDateTime.now(ZoneOffset.UTC)));

        terminateSessions(u.getUserId());
    }

    @Transactional
    public void resetPassword(String targetUserId, String actorUserId, String clientIp) {
        User u = userRepo.findByUserId(targetUserId).orElseThrow(() -> new NotFoundException("user not found"));
        String temp = generateTempPassword(16);
        u.setPasswordHash(encoder.encode(temp));
        u.setForceChangePw(true);
        u.setFailedAttempts(0);
        if (u.getStatus() == UserStatus.LOCKED) u.setStatus(UserStatus.ACTIVE);
        u.setLockedAt(null);
        userRepo.save(u);

        audit.log(new AuditEvent(actorUserId, AuditAction.USER_PASSWORD_RESET, "USER",
                String.valueOf(u.getId()), null, null,
                "admin password reset", clientIp, OffsetDateTime.now(ZoneOffset.UTC)));

        email.sendPasswordReset(u.getEmail(), u.getUserId(), temp);
    }

    private Set<Role> resolveRoles(List<String> roleCodes) {
        Set<Role> out = new LinkedHashSet<>();
        for (String code : roleCodes) {
            Role r = roleRepo.findByRoleCode(code).orElseThrow(() ->
                    new UnprocessableEntityException("USER_003", "unknown role: " + code));
            out.add(r);
        }
        return out;
    }

    private void terminateSessions(String userId) {
        for (Object principal : sessionRegistry.getAllPrincipals()) {
            if (principal == null) continue;
            String name = principal.toString();
            if (name.equals(userId)) {
                for (SessionInformation si : sessionRegistry.getAllSessions(principal, false)) {
                    si.expireNow();
                }
            }
        }
    }

        return sb.toString();
    }
```

```java
@Component
public class AuditorExpirySessionScheduler {

    private final UserRepository userRepo;
    private final AuditService audit;
    private final SessionRegistry sessionRegistry;

    public AuditorExpirySessionScheduler(UserRepository userRepo, AuditService audit,
                                         SessionRegistry sessionRegistry) {
        this.userRepo = userRepo;
        this.audit = audit;
        this.sessionRegistry = sessionRegistry;
    }

    /** Daily at 00:00 KST: disable any user whose valid_until ≤ today. BR-USER-015. */
    @Scheduled(cron = "0 0 0 * * *", zone = "Asia/Seoul")
    @Transactional
    public void runExpiryTick() {
        List<User> expired = userRepo.findExpiredAuditors();
        for (User u : expired) {
            u.setStatus(UserStatus.DISABLED);
            userRepo.save(u);
            terminate(u.getUserId());
            audit.log(new AuditEvent(null, AuditAction.AUDITOR_EXPIRED, "USER",
                    String.valueOf(u.getId()), null, null,
                    "valid_until reached: " + u.getValidUntil(),
                    null, OffsetDateTime.now(ZoneOffset.UTC)));
        }
    }

    private void terminate(String userId) {
        for (Object p : sessionRegistry.getAllPrincipals()) {
            if (p == null || !p.toString().equals(userId)) continue;
            for (SessionInformation si : sessionRegistry.getAllSessions(p, false)) si.expireNow();
        }
    }
}
```

```java
package com.lab.edms.auth;

import com.lab.edms.user.User;
import com.lab.edms.user.UserRepository;
import com.lab.edms.user.UserStatus;
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;
import org.springframework.stereotype.Component;
import org.springframework.transaction.annotation.Transactional;

import java.time.LocalDate;
import java.time.OffsetDateTime;
import java.util.Optional;

@Component
public class LocalAuthProvider implements AuthProvider {

    static final int MAX_FAILED_ATTEMPTS = 5;

    private final UserRepository userRepo;
    private final BCryptPasswordEncoder encoder;

    public LocalAuthProvider(UserRepository userRepo, BCryptPasswordEncoder encoder) {
        this.userRepo = userRepo;
        this.encoder = encoder;
    }

    @Override
    public boolean supports(String providerCode) { return "LOCAL".equals(providerCode); }

    @Override
    @Transactional
    public AuthResult authenticate(String userId, String rawPassword, String clientIp) {
        Optional<User> opt = userRepo.findByUserIdWithRoles(userId);
        if (opt.isEmpty()) {
            return new AuthResult.InvalidCredentials(MAX_FAILED_ATTEMPTS - 1);
        }
        User u = opt.get();

        if (u.getStatus() == UserStatus.DISABLED) return new AuthResult.AccountDisabled();
        if (u.getStatus() == UserStatus.LOCKED)   return new AuthResult.AccountLocked();

        // BR-USER-014/015 validity window
        LocalDate today = LocalDate.now();
        if (u.getValidFrom() != null && today.isBefore(u.getValidFrom())) return new AuthResult.AccountDisabled();
        if (u.getValidUntil() != null && !today.isBefore(u.getValidUntil().plusDays(1))) {
            return new AuthResult.AccountDisabled();
        }

        if (u.getPasswordHash() == null) return new AuthResult.InvalidCredentials(MAX_FAILED_ATTEMPTS - 1);

        if (!encoder.matches(rawPassword, u.getPasswordHash())) {
            int next = u.getFailedAttempts() + 1;
            u.setFailedAttempts(next);
            if (next >= MAX_FAILED_ATTEMPTS) {
                u.setStatus(UserStatus.LOCKED);
                u.setLockedAt(OffsetDateTime.now());
                userRepo.save(u);
                return new AuthResult.AccountLocked();
            }
            userRepo.save(u);
            return new AuthResult.InvalidCredentials(MAX_FAILED_ATTEMPTS - next);
        }

        u.setFailedAttempts(0);
        u.setLastLoginAt(OffsetDateTime.now());
        userRepo.save(u);

        if (u.isForceChangePw()) return new AuthResult.ForcePasswordChange(u);
        return new AuthResult.Success(u);
    }
}
```
